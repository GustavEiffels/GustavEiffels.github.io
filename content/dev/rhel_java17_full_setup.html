[RHEL] Java 17 설치 가이드

2026.04.22 Java 17 · RHEL 8 · Airgap · Online
① 파일 이관›② JVM 디렉토리 생성› ③ 압축 해제›④ 환경변수 설정›⑤ 검증
1
Docker Compose 구성 (UBI 컨테이너 환경)

RHEL의 공식 이미지인 Universal Base Image (UBI)를 활용합니다.

docker-compose.yml 
version: "3.8"

services:
  rhel-test:
    image: registry.access.redhat.com/ubi8/ubi:latest
    container_name: rhel-test-server
    tty: true
    volumes:
      - .:/test_dir
컨테이너 기반이 아닌 실서버라면 이 단계는 생략합니다.
2
이관 파일 준비

Oracle 또는 Eclipse Temurin JDK 17 Linux 바이너리 tar.gz 파일을 외부망 PC에서 다운로드한 뒤 폐쇄망 서버로 이관합니다.

이관 파일: jdk-17.0.12_linux-x64_bin.tar.gz

다운로드 소스 (외부망 PC에서)
Oracle JDK: oracle.com/java/technologies/downloads/#java17
Eclipse Temurin (무료): adoptium.net/temurin/releases/?version=17
3
JVM 디렉토리 생성 및 압축 해제

표준 JVM 경로(/usr/lib/jvm)에 JDK를 배치하여 관리 효율성을 높입니다.

Terminal 
# JVM 디렉토리 생성
> mkdir -p /usr/lib/jvm

# 지정 경로에 압축 해제
> tar -xvf jdk-17.0.12_linux-x64_bin.tar.gz -C /usr/lib/jvm/

# 해제된 디렉토리명 확인
> ls /usr/lib/jvm/
✅ /usr/lib/jvm/jdk-17.0.12/ 디렉토리가 생성되면 정상입니다.
4
환경변수 구성 (전역 적용)

RHEL의 /etc/profile.d/에 스크립트를 배치하면 모든 유저에게 자동으로 적용됩니다.

Bash Shell 
# JAVA_HOME 및 PATH 전역 등록
> echo 'export JAVA_HOME=/usr/lib/jvm/jdk-17.0.12' > /etc/profile.d/java.sh
> echo 'export PATH=$JAVA_HOME/bin:$PATH' >> /etc/profile.d/java.sh

# 현재 세션에 즉시 적용
> source /etc/profile.d/java.sh

# 설정 확인
> echo $JAVA_HOME
⚠️ 실제 해제된 디렉토리명과 JAVA_HOME 경로가 일치해야 합니다. ls /usr/lib/jvm/으로 이름을 확인하세요.
5
설치 검증
> java -version
> javac -version
> echo $JAVA_HOME
✅ 성공 출력
openjdk version "17.0.12" ...
javac 17.0.12
① DNF 설치›② JAVA_HOME 자동 감지› ③ 환경변수 설정›④ 다중 버전 관리›⑤ 검증
1
OpenJDK 17 DNF 설치

RHEL 8 AppStream 리포지토리에 포함된 OpenJDK 17을 DNF로 설치합니다. 별도 리포지토리 추가 없이 즉시 설치 가능합니다.

root@server 
# JRE + JDK (컴파일러 포함) 설치
> dnf install -y java-17-openjdk java-17-openjdk-devel

# 설치된 Java 목록 확인
> alternatives --list | grep java
패키지 구분
java-17-openjdk — JRE (런타임만)
java-17-openjdk-devel — JDK (컴파일러 javac 포함)
2
JAVA_HOME 자동 감지 및 환경변수 설정

DNF 설치 후 실제 JDK 경로를 readlink로 자동 감지하여 환경변수를 설정합니다.

Bash Shell 
# JAVA_HOME 경로 자동 감지
> JAVA_HOME=$(dirname $(dirname $(readlink -f $(which java))))
> echo "감지된 JAVA_HOME: $JAVA_HOME"

# 전역 환경변수 등록
> echo "export JAVA_HOME=$JAVA_HOME" > /etc/profile.d/java.sh
> echo 'export PATH=$JAVA_HOME/bin:$PATH' >> /etc/profile.d/java.sh

# 현재 세션에 적용
> source /etc/profile.d/java.sh
3
다중 JDK 버전 관리 (선택)

서버에 여러 Java 버전이 설치된 경우 alternatives로 기본 버전을 지정합니다.

# 기본 Java 버전 선택 (대화형)
> alternatives --config java
