[Tomcat] 테스트용 ROOT.war 생성 및 배포 가이드

2026.04.21 Middleware Utility

1. 웹 콘텐츠 작성 (HTML)

배포 성공 시 브라우저에 표시될 기본 페이지를 작성합니다. index.html이라는 파일명으로 저장합니다.

index.html 
<!DOCTYPE html>
<html>
<body>
    <h1>✅ Tomcat Deployment Success</h1>
    <p>The server is running perfectly!</p>
</body>
</html>

2. WAR 패키징 (압축)

HTML 파일을 ZIP 구조로 압축한 뒤 확장자를 .war로 변경하여 Tomcat이 인식할 수 있게 합니다.

Windows PowerShell

PowerShell 
PS> Compress-Archive -Path ./index.html -DestinationPath ./ROOT.zip
PS> Rename-Item -Path ./ROOT.zip -NewName ROOT.war

Linux Bash

Bash Shell 
$ zip ROOT.war index.html

3. Tomcat 서버 배포

생성된 ROOT.war를 Tomcat의 자동 배포 경로(webapps)로 복사합니다. 기존 ROOT 폴더를 먼저 삭제해야 새 파일이 즉시 반영됩니다.

Terminal 
# 1. 기존 ROOT 디렉토리 제거 (필수)
> rm -rf /opt/tomcat/webapps/ROOT

# 2. 새 WAR 파일 복사
> cp ROOT.war /opt/tomcat/webapps/
