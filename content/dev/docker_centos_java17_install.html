[Docker] CentOS 7 Java 17 수동 설치 가이드

2026.04.21 Technical Guide

1. Docker Compose 구성

로컬 JDK 파일을 공유하기 위한 볼륨 마운트가 포함된 전체 설정입니다.

docker-compose.yml 
version: "3.8"

services:
  centos-test:
    image: centos:7
    container_name: centos-test-server
    tty: true
    stdin_open: true
    volumes:
      - .:/test_dir
    working_dir: /test_dir
    restart: always

2. 압축 해제 및 경로 지정

표준 JVM 디렉토리에 JDK 바이너리를 배치합니다.

Terminal 
# 1. JVM 디렉토리 생성
> mkdir -p /usr/lib/jvm

# 2. 지정 경로에 압축 해제
> tar -xvf jdk-17.0.12_linux-x64_bin.tar.gz -C /usr/lib/jvm/

3. 환경 변수 자동화 설정

echo 명령어를 활용해 재부팅 후에도 유지되는 환경 변수를 등록합니다.

Bash Shell 
# JAVA_HOME 등록
> echo "export JAVA_HOME=/usr/lib/jvm/jdk-17.0.12" > /etc/profile.d/java.sh

# PATH 등록
> echo "export PATH=\$JAVA_HOME/bin:\$PATH" >> /etc/profile.d/java.sh

# 즉시 적용
> source /etc/profile.d/java.sh

4. 최종 검증

Java 엔진이 정상적으로 로드되었는지 확인합니다.

> java -version

# 출력: java version "17.0.12" ...
